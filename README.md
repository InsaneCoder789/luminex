# Luminex | Custom Glowing Art

Luminex is a premium boutique design studio specializing in the intersection of light and memory. We transform your most cherished high-resolution photographs into enchanting, etched works of art that illuminate space with atmospheric warmth.

## 🏺 The Collection

- **Luminex M1 (Flagship)**: Our most ambitious engineering project featuring custom prism optics and a liquid titanium base.
- **LumiFrame V1 (Desktop)**: A minimalist take on memory, perfect for modern workspaces.
- **LumiBox (Premium)**: Deep-etched memories housed in a handcrafted hardwood case.
- **LumiChain (Artifact)**: A portable pocket-sized memory handcrafted with brass fittings.

## ✨ Experience Design

The application is built with a focus on "Architectural Luxury" and "Atmospheric Interaction":
- **Cinematic Foundations**: Immersive video backgrounds and glassmorphic UI layers.
- **Micro-interactions**: GSAP and Framer Motion driven state transitions that mimic the fluidity of light.
- **Artisan Checkout**: A bespoke allocation request flow designed to feel as premium as the products themselves.
- **Responsive Precision**: Fully optimized for mobile collectors and desktop galleries.

## 🛠 Tech Stack

- **Framework**: React 18+ with Vite
- **Styling**: Tailwind CSS (Custom Gold & Glow Theme)
- **Animation**: GSAP (ScrollTrigger) & Framer Motion
- **Icons**: Lucide React
- **Typography**: Inter & Serif pairings for an editorial feel

## 🌿 Philosophy

Every Luminex piece is a "Donum Original"—a gift intended to last a lifetime. We combine AI-assisted precision etching with 48 hours of artisanal hand-finishing in our studio in India.
